## 단순 알고리즘 예시: nearest neighbor
1. train function
단순히 train 데이터를 메모리에 저장하는 과정.
- 이미지 비교 distance metric
  - L1(Manhattan) distance: 차이의 절댓값의 합
  - L2(Euclidean) distance: 차이의 제곱의 합의 root
- O(1)
1. predict function
testing sample간의 거리 계산
O(N)
=> 너무 비효율적
