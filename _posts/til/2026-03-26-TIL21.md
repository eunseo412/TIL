### dynamic allocation
  
new
- 어떤 타입인지, 몇개의 공간인지 명시 해야함

delete
- `delete [] 포인터명`의 형태로 삭제해야함
- delete하면 arr(아래 코드에서)가 os로 넘어감?? 뭔소리지
- 포인터가 사라져서 arr에 접근할 수 없을니까?

```cpp
int main()
{
    int *arr;
    int size;
    cin >> size;

    // dynamic allocation
    arr = new int[size];

    arr[0] = 10;
    // *(arr+1) = 10;
    arr[1] = 3;

    delete[] arr;
}
```

### 변수형
- 정적 캐스팅 : 변수 선언시 타입을 명시하야 한다.
- 정수형 int, 실수형 double, 논리형 bool, 문자형 char, short, long, long long, float 등
- 변수형(변수의 타입) / 변수이름 / 초기값(선택가능)
- 선언한 타입에 따라 저장할 수 있는 값의 종류와 크기가 결정된다.
-  4byte = 32-bite
  
### 변수의 초기화
- 초기화되지 않은 변수 : 쓰레기값이 들어있다.
`int uninitializedl`
- 초기값을 가진 변수 -> 변수 선언시 실수를 줄이기 위해 초기값 넣어주는게 좋다.
`int initialized = 100;`
  
### 변수의 크기 확인
- 타입에 따라 변수의 크기가 다르다.
- sizeof 연산자로 확인

### 배열
- 동일한 타입의 변수를 연속적인 메모리 공간에 저장 가능
- 인덱스을 통해 특정 위치 원소에 빠르게 접근
- 배열의 크기는 선언 시에 지정되고 이후에 크기 변경 불가능
- 임의 위치에 원소 삽입하는 경우 O(N) - 기존의 원소를 한칸씩 뒤로 이동시켜야 하기 때문
- 맨 뒤에 원소를 삽입하는 경우 O(1)
- 배열의 타입 / 배열의 이름 / 배열의 크기
`int a[5]`

### 문자열
- `#include <string>`
- c++의 STL에서 제공하는 문자열 클래스
- '\0'문자로 종료되지 않는다.
- +를 이용해서 문자열을 연결할 수 있다.
- []로 임의의 문자에 접근 가능하다.
  
```cpp
#include <string>
string str1 = "Hello, world!";
string str2 = str1.substr(0, 5);    // "Hello"
string str3(5, 'A');    // "AAAAA"
string str4("Apple");
string str5(str2 + ", " + "World");     //"Hello, World"
string str6 = str1;
```
